Only `Select.Root` is wrapped, to inject the `size` recipe's `data-size`; every
other part is Ark's verbatim, styled by `components.css` keyed on Ark's
`data-part` plus `data-highlighted`/`data-state`. The menu renders in a
`Portal`, unmounted while closed, so SSR/hydration stay clean.

<CodeBlock
  code={`<Select.Root collection={frameworks} size="md">
  <Select.Label>Framework</Select.Label>
  <Select.Trigger>
    <Select.ValueText placeholder="Select a framework" />
  </Select.Trigger>
  <Portal>
    <Select.Positioner>
      <Select.Content>
        {frameworks.items.map((item) => (
          <Select.Item key={item.value} item={item}>
            {item.label}
          </Select.Item>
        ))}
      </Select.Content>
    </Select.Positioner>
  </Portal>
</Select.Root>`}
/>
