Dialog composes as `Root > Trigger + Portal(> Backdrop + Positioner > Content)`.
Ark drives focus trap, focus restore, scroll lock, and the `aria-labelledby`/
`aria-describedby` wiring; content is unmounted while closed, so an SSR render
emits only the trigger. No CVA variants — the surface comes straight from the
`--popover` / `--popover-foreground` token slots via `components.css`.

  code={`<Dialog.Root>
  <Dialog.Trigger>Open</Dialog.Trigger>
  <Portal>
    <Dialog.Backdrop />
    <Dialog.Positioner>
      <Dialog.Content>
        <Dialog.Title>Delete item</Dialog.Title>
        <Dialog.Description>This can't be undone.</Dialog.Description>
      </Dialog.Content>
    </Dialog.Positioner>
  </Portal>
</Dialog.Root>`}
