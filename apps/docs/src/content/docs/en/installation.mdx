Pick the binding for your framework. Every binding depends on `@moderno/core`
(the recipes) and is painted by `@moderno/css` (the shared stylesheet).

## React

<Install pkg="@moderno/react" />

## Import the stylesheet once

Import `@moderno/css` a single time at your app root — it carries the component
styles and the default theme's contract slots.

```ts
import "@moderno/css";
```

## Scaffold from the registry (optional)

You can also eject source with the CLI, which reads the registry served at
`/r/registry.json`.
