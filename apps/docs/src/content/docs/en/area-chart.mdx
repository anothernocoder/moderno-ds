AreaChart is a deterministic map from a model built by `@moderno/charts-core` to
`<path>` elements — a filled area plus a stroked top line per series. It bakes
in no colour — each series paints from `--chart-1…5` keyed on its index, so it
re-themes with the rest of the system.
