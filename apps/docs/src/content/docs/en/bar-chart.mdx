BarChart is a deterministic map from a model built by `@moderno/charts-core` to
`<rect>` elements — one bar per category per series, positioned on a band
scale. It bakes in no colour — each series paints from `--chart-1…5` keyed on
its index, so it re-themes with the rest of the system.
