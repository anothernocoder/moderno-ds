Moderno ofrece un único contrato de estilos — slots de tokens semánticos
pintados por una sola hoja de estilos compartida — y enlaces ligeros para cada
framework. Los componentes no llevan color propio; remapear los slots vuelve a
tematizar todo a la vez.

Explora los componentes en la barra lateral, o abre el **Editor de temas** para
diseñar tu propia marca y exportar un `theme.css` y un `tokens.dtcg.json`.
