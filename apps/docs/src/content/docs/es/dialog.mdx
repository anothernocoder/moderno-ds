Dialog se compone como `Root > Trigger + Portal(> Backdrop + Positioner >
Content)`. Ark maneja el atrapado y restauración de foco, el bloqueo de
scroll y la conexión `aria-labelledby`/`aria-describedby`; el contenido no se
monta mientras está cerrado, así que un render SSR solo emite el trigger. Sin
variantes de CVA — la superficie viene directo de los slots `--popover` /
`--popover-foreground` vía `components.css`.

<CodeBlock
  code={`<Dialog.Root>
  <Dialog.Trigger>Abrir</Dialog.Trigger>
  <Portal>
    <Dialog.Backdrop />
    <Dialog.Positioner>
      <Dialog.Content>
        <Dialog.Title>Eliminar elemento</Dialog.Title>
        <Dialog.Description>Esto no se puede deshacer.</Dialog.Description>
      </Dialog.Content>
    </Dialog.Positioner>
  </Portal>
</Dialog.Root>`}
/>
