BarChart es un mapeo determinista desde un modelo construido por
`@moderno/charts-core` hacia elementos `<rect>` — una barra por categoría por
serie, posicionada sobre una escala de bandas. No incluye color propio — cada
serie se pinta con `--chart-1…5` según su índice, así se re-tematiza con el
resto del sistema.
