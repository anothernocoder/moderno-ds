AreaChart es un mapeo determinista desde un modelo construido por
`@moderno/charts-core` hacia elementos `<path>` — un área rellena más una línea
superior trazada por serie. No incluye color propio — cada serie se pinta con
`--chart-1…5` según su índice, así se re-tematiza con el resto del sistema.
