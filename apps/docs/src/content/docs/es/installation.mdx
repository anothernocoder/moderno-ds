Elige el enlace para tu framework. Cada enlace depende de `@moderno/core` (las
recetas) y lo pinta `@moderno/css` (la hoja de estilos compartida).

## React

<Install pkg="@moderno/react" />

## Importa la hoja de estilos una vez

Importa `@moderno/css` una sola vez en la raíz de tu app — incluye los estilos
de los componentes y los slots del contrato del tema por defecto.

```ts
import "@moderno/css";
```

## Generar desde el registro (opcional)

También puedes extraer el código fuente con la CLI, que lee el registro servido
en `/r/registry.json`.

<Install pkg="@moderno/cli" mode="cli" />

## Sembrar AGENTS.md para agentes de código (opcional)

`moderno init` también escribe un stanza en `AGENTS.md` — las reglas de oro,
más instrucciones para usar el servidor MCP `moderno` y ejecutar
`validate_usage` antes de terminar. Pasa `--claude` para además escribir un
`CLAUDE.md` gemelo; cualquiera de los dos archivos se fusiona en el lugar en
ejecuciones repetidas, sin sobrescribir el resto de tu contenido.

```sh
npx @moderno/cli init --claude
```

Consulta [Usar Moderno con agentes](/es/using-with-agents) para ver qué dice
el stanza y cómo funciona realmente el ciclo del MCP al que apunta.
